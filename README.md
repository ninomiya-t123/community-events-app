# 地域コミュニティイベント管理アプリ

## 📌 プロジェクト概要
本アプリは、地域住民が気軽に地域イベント情報を閲覧・共有できることを目的としています。  
都市部での人口集中やライフスタイルの多様化により地域のつながりが希薄化している中、  
イベントを通じて交流を促進し、地域活性化を目指します。  

---

## 🛠 使用技術
- **React 18** <br>
  コンポーネントベースで画面を構築し、状態管理に Hooks を利用。
- **React Router DOM** <br>
  ログイン画面や管理画面などの画面遷移を実装。
- **JavaScript (ES6+)** <br>
  イベントの追加・編集・削除・検索・承認などのロジックを実装。
- **CSS** <br>
  初期のスタイルやモーダル表示用の基本デザインを作成。
- **Tailwind CSS** <br>
  ボタン・フォーム・テーブル・レイアウトのスタイリングを効率的に適用。

## 📱 どのようなアプリか

- **Who（ユーザー）：** <br>
（管理者として）地域イベント管理者<br>
（参加者・主催者として）地域住民

- **Where（どこで）：** <br>
一家に一台、車があるぐらいの郊外地域で使われるアプリ

- **When（いつ）：** <br>
（管理者）毎日、毎週末など定期的に管理できる<br>
（参加者）いつでも手軽にイベントを参照できる<br>
（主催者）いつでも手軽にイベントを申請できる

- **What（なにを実現できるアプリか）：** <br>
地域で交流できる場づくり・管理

- **Why（なぜこのアプリを使いたいのか）：**
1. 多様な世代が交流できる場が欲しいから<br>
  「地域行事への参加率は、若い世代のほうがシニア世代よりも低いのではないか」と仮定。<br>
  まずは若い世代の参加率をあげることを第一に考え、手軽・シンプルなUIにした。<br>
  将来的には、シニア世代へのアプローチも考え、段階的に地域活性化を実現したい。<br>

2. もっと気軽に地域のイベント情報を知りたいから<br>
  「気軽に」を「利用のハードルが低い」と解釈し、アカウントを作らなくてもまずは参照ができるようにした。<br>

3. 自分のスキルや経験を地域で活かしたいから<br>
  個人のスキルや経験を活かすためのイベントを、誰でも申請可能とした。<br>

- **How（どのように使うか）：** <br>
ブラウザからアクセスして利用する。
将来的にスマホ対応を実現し、スマホアプリとブラウザの両方から利用できるようにしたい。
※利用者目線だと、アプリを入れるメリットがあまりないため手軽にブラウザ利用したい。
　ただ、管理者目線だと日々申請情報をスマホアプリで手軽に確認できると便利だと考えた。

## ✨ 機能の概要
- **イベント一覧の表示（タイトル・日付・場所で検索可能）**
- **イベント詳細の確認（一覧から遷移可能）**
- **イベントの申請・承認（一般ユーザーによる申請、管理者による承認が可能）**

## 🚀 セットアップ手順

```bash
cd community-events-app

# パッケージをインストール
npm install

# 開発サーバーを起動
npm start

```
