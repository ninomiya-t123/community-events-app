# 地域コミュニティイベント管理アプリ

## 📌 プロジェクト概要
本アプリは、地域住民が気軽に地域イベント情報を閲覧・共有できることを目的としています。  
都市部での人口集中やライフスタイルの多様化により地域のつながりが希薄化している中、  
イベントを通じて交流を促進し、地域活性化を目指します。  

---

## 🛠 使用技術
- **React 18**
- **React Router DOM**
- **JavaScript (ES6+)**
- **CSS**

---

## 🚀 セットアップ手順

```bash
# リポジトリをクローン
git clone https://github.com/ninomiya-t123/community-events-app.git
cd community-events-app

# パッケージをインストール
npm install

# 開発サーバーを起動
npm start

```

## 📱 どんなアプリか（5W1H）

■Who（ユーザー）：<br>
（管理者として）イベント管理者<br>
（参加者・主催者として）地域住民

■Where（どこで）：<br>
一家に一台、車があるぐらいの郊外地域で使われるアプリ

■When（いつ）：<br>
（管理者）毎日、毎週末など定期的に管理できる<br>
（参加者）いつでも手軽にイベントを参照できる<br>
（主催者）いつでも手軽にイベントを申請できる

■What（なにを実現できるアプリか）：<br>
地域で交流できる場づくり・管理

■Why（なぜこのアプリを使いたいのか）<br>
①多様な世代が交流できる場が欲しいから<br>
　[私の解釈]<br>
　　実体験から、「地域行事への参加率は、若い世代のほうがシニア世代よりも低いのではないか」と<br>
　　仮定し、まずは若い世代の参加率をあげることを第一に考えた。<br>
　　そのため、"タイパ"を意識し、手軽・シンプルなUIにした。<br>
　　将来的には、シニア世代へのアプローチも考えることで、段階的に地域活性化を実現したい。

②もっと気軽に地域のイベント情報を知りたいから<br>
　[私の解釈]<br>
　　「気軽に」を、「アカウントを作らなくてもまずは参照ができるなど、<br>
　　利用のハードルが低い」と解釈した。

③自分のスキルや経験を地域で活かしたいから<br>
　[私の解釈]<br>
　誰でも気軽に自分のスキルや経験を活かすためのイベントを申請できるようにすることで、<br>
　実現できると考えた。

■How（どのように使うか）<br>
ブラウザからアクセスして利用する。

## ✨ 機能の概要
- **イベント一覧の表示（タイトル・日付・場所で検索可能）**
- **イベント詳細の確認（一覧から遷移可能）**
- **イベントの申請・承認（一般ユーザーによる申請、管理者による承認が可能）**
